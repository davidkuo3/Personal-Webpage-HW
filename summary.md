# 郭子熠個人網頁專案總結報告

## 1. 專案概述
本專案為 **郭子熠** 打造了一個現代化、單頁式的個人主頁，旨在展示極簡且具科技感的視覺設計，並具備實時時間顯示功能。

## 2. 核心功能
- **實時時鐘 (Live Clock)**：每秒自動更新的 24 小時制時鐘（時:分:秒）。
- **完整日期顯示**：自動顯示當前的星期、月份、日期與年份（英文格式）。
- **姓名動態效果**：姓名「郭子熠」具有紫色漸變流光（Shimmer）動畫。
- **Streamlit 整合**：透過 `app.py` 將 HTML 內容嵌入 Streamlit，方便後續快速發布。

## 3. 視覺設計 (Glassmorphism)
專案採用了 **毛玻璃 (Glassmorphism)** 設計風格，包含以下亮點：
- **動態背景**：深紫色與靛藍色的 400% 縮放漸變背景，維持流暢的動態感。
- **懸浮球體**：背景中配置了三個大尺寸的高斯模糊懸浮球，營造層次深度。
- **毛玻璃卡片**：中心卡片具有高度背景模糊 (backdrop-filter) 與半透明邊框。
- **交互動畫**：
    - **進場動畫**：卡片與內容採用錯落的淡入位移效果。
    - **懸停特效**：滑鼠懸停於卡片時會產生向上浮起的動態反饋。
- **現代字體**：使用 Google Fonts 的 **Outfit** 作為標題與時鐘字體，**Inter** 作為輔助字體。

## 4. 技術架構
- **前端工具**：HTML5, Vanilla CSS, JavaScript (ES6+)。
- **後端整合**：Python Streamlit (用於網頁部署與展示)。
- **版本控制**：使用 Git 進行管理並推送至 GitHub 儲存庫。

## 5. 文件說明
- `index.html`：包含所有的結構、樣式與 JavaScript 邏輯。
- `app.py`：Streamlit 入口文件，負責渲染 HTML。
- `summary.md`：專案進度與功能說明文件。
